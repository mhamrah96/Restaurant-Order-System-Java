# Restaurant Order System

A Java console program that simulates a restaurant ordering system.  
Users can add or remove items, view their current order, and calculate the total cost.

## Features
- Display a restaurant menu with 10 items.
- Add items to the order and track quantities.
- Remove items from the order.
- View current order with item quantities and prices.
- Calculate total cost of the order.

## Technologies Used
- Java
- Arrays
- Loops and conditionals
- Scanner for user input
